📝 وصف المشروع (Project Overview)
هذا المشروع هو تطبيق  لسطح المكتب يمثل نظام خبير طبي (Medical Expert System). يقوم التطبيق بتحليل القياسات الخلوية المستخرجة من الفحوصات المجهرية لتحديد طبيعة الأورام (حميدة أو خبيثة) بدقة عالية، وذلك باستخدام خوارزميات التعلم الآلي المتقدمة. يهدف النظام إلى تقليل نسبة الخطأ البشري وسرعة اتخاذ القرار الطبي.

✨ مميزات البرنامج (Key Features)
محرك ذكاء اصطناعي حقيقي: يعتمد النظام على خوارزمية Random Forest Classifier المدربة على قاعدة بيانات "ويسكونسن" العالمية، مما يضمن نتائج مبنية على حالات طبية حقيقية.

واجهة مستخدم طبية (Modern UI/UX): واجهة عصرية وسهلة الاستخدام مصممة بمكتبة CustomTkinter لتدعم الوضع الداكن (Dark Mode) وتوفر تجربة مريحة للأطباء.

معالجة ذكية للأخطاء: النظام مزود بآلية (Robustness) تمنع الانهيار عند إدخال بيانات خاطئة وتوجه المستخدم عبر رسائل تحذيرية ذكية.

نظام التقارير الشامل: لا يكتفي البرنامج بالنتيجة فقط، بل يعرض "نسبة الثقة" (Confidence Score) في التشخيص، مما يعزز من مصداقية النتائج أمام اللجنة العلمية.
البيانات المدخلة (The 12 Inputs)
